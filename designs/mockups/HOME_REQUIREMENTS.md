# Требования для главной страницы (по макету)

## Header
- Логотип "Минтруд России" слева (иконка + текст)
- Текст "Центр Мониторинга Минтруда" по центру
- Справа: зелёная точка + "обновление: ЧЧ:ММ:СС" + иконка темы

## Hero section
- Синий градиент (135deg, #1d4ed8 → #3b82f6)
- Текст слева: "Мониторинговые сервисы Минтруда России" + описание
- Иллюстрация/картинка справа (шириной ~30% блока)
- Справа две карточки:
  - "Данные онлайн. Обновление каждые 5 секунд"
  - "Провал по клику на любой блок"

## Карточки центров
- 3 колонки
- Белые карточки (светлая тема) / тёмные (тёмная тема)
- Синие иконки (24px)
- Кнопка → справа
- Активная карточка: синяя рамка и точка

## Цвета (из макета)
- Основной синий: #2563eb
- Белый фон карточек: #ffffff (light) / #3d4555 (dark)
- Текст: чёрный (light) / белый (dark)
