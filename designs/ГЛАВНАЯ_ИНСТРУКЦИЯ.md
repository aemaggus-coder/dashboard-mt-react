# Главная страница - Инструкция

## 📋 Макеты (положены в проект)

- ✅ `designs/mockups/Мониторинг_главная_light.pdf` - Светлая тема
- ✅ `designs/mockups/Мониторинг_главная (dark 2).pdf` - Тёмная тема

## 🎨 Реализация

Главная страница реализована в `src/pages/Home.jsx` и соответствует **обоим макетам**:

### ✅ Light тема (светлый фон)
- Белые карточки центров
- Синие иконки
- Светлый фон
- **Нажми на иконку темы** → переходит на Dark

### ✅ Dark тема (тёмный фон)
- Тёмные карточки с border
- Синие иконки
- Тёмный фон
- **Нажми на иконку темы** → переходит на Light

## 📸 Элементы на главной

| Элемент | Статус | Заметки |
|---------|--------|---------|
| Header (логотип + время) | ✅ | Зелёная точка обновления работает |
| Hero section | ✅ | С иллюстрацией |
| 2 инфо карточки | ✅ | Данные онлайн + Провал по клику |
| 12 карточек центров | ✅ | С синими иконками и кнопками |
| Тёмная/светлая тема | ✅ | Работает переключение |
| CSV Export | ✅ | Кнопка в дашборде |
| Live данные | ✅ | Обновление каждые 5 сек |

## 🖼️ Если нужна реальная иллюстрация вместо SVG

1. Возьми изображение (PNG, SVG, JPG)
2. Положи в: `public/assets/images/hero-illustration.png`
3. Замени в коде (в `src/pages/Home.jsx`):

```jsx
// Вместо SVG:
<img 
  src="/assets/images/hero-illustration.png" 
  alt="Мониторинговые сервисы"
  style={{ width: '240px', height: '200px', objectFit: 'contain' }}
/>
```
